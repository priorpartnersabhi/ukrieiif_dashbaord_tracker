Subject: Place+ Dashboard Tracker — add ideas & vote for what matters

Hi all,

I've set up a shared tracker for Place+ dashboards so we can collect ideas, see what's in progress, and vote on what to build next.

Link: https://priorpartnersabhi.github.io/ukrieiif_dashbaord_tracker/
Password: placeplus2026

How to use it

1. Open the link and enter your name and the site password above.
2. You'll see the current list of dashboards, their theme, status, and how many votes each has.

Adding a new dashboard idea

Click "+ Add dashboard" in the top right and fill in:

- Dashboard name — what you'd call it
- Theme — pick one from the dropdown (People, Place, Housing, Environment, Health, Movement, Infrastructure)
- Data source — paste the link to the dataset if you have one, or just describe it in words (I'll replace this with the dashboard link once it's built)
- Notes — a short description of what the dashboard should do / who it's for

Voting

Next to each dashboard there's a small ▲ button with a count. Click it to vote — click again to un-vote. One vote per person per dashboard. I'll use votes to help prioritise what to build first.

Status icons

You'll see a coloured circle next to each dashboard:

- Not started
- In progress
- Data not available
- Completed

Feel free to click through and update the status if you know something has changed.

Shout if anything breaks or feels off.
